# Firebase-Image-Upload

This is java code which describes how you can upload images from android application and store it on Firebase Cloud Storage.

First you need to set up the firebase project
or you can just connect your application with firebase realtime database.
For this you can follow the steps below:
1. Open Android studio
2. Select 'Tools' from the menu.
    then a sider will be opened.
3. select 'Realtime database' and connect the project from the Firecbase account.
    also connect authentication & cloud messaging.
4. then copy the following code in the Android studio.

Add Internet permission in the manifest file.

Add folowing dependencies in your project's : 'build.gradle'
    implementation 'com.android.support:cardview-v7:27.1.1'
    implementation 'com.android.support:recyclerview-v7:27.1.1'
    implementation 'com.squareup.picasso:picasso:2.5.2'
Remember that support veersion should be same for all support implementation.

Copy the java files into your project 'package' folder.

Copy the layout files in your project 'layout' folder.

Now Run
